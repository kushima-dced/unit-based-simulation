## 画面1: トップ / シミュレーション (`/`)

### 構成

```
┌─────────────────────────────────────────────────────────┐
│  ヘッダー（アプリ名）                                      │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  【投資条件入力エリア】                                    │
│  ┌─────────────────────────────────────────────────┐   │
│  │ ・目標資産額（万円）、目標達成年数                    │   │
│  │ ・追加投資する年（最大4つ、比較用、0=開始時）         │   │
│  │ ・積立投資: する/しない                             │   │
│  │ ・価格単位: 1口1円 / 1口1万円                        │   │
│  │ ・商品（最大4つ）: 名前、現在価格、期待成長率        │   │
│  │   保有口数 or 保有評価額（円）、目標口数比率          │   │
│  │   積立する場合: 月々の積立金額（円）                  │   │
│  │ [シミュレーション実行] ボタン                        │   │
│  └─────────────────────────────────────────────────┘   │
│                                                         │
│  【シミュレーション結果エリア】                            │
│  ┌─────────────────────────────────────────────────┐   │
│  │ ・シナリオ比較（複数指定時）                        │   │
│  │ ・目標達成可否、目標との差                          │   │
│  │ ・合計サマリー（累計投資額、保有口数、評価額等）      │   │
│  │ ・商品別サマリー                                    │   │
│  │ ・推移グラフ（年別）                                │   │
│  │ ・詳細テーブル（年別・合計、〇年目形式）              │   │
│  │ ・詳細（年別・商品毎: 積立金額、商品価格含む）        │   │
│  └─────────────────────────────────────────────────┘   │
│                                                         │
└─────────────────────────────────────────────────────────┘
```
