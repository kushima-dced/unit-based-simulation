# 技術スタック

## 概要

本プロジェクトで採用している技術一覧です。

---

## フロントエンド

| 技術 | バージョン | 用途 |
|------|-----------|------|
| **Next.js** | 16.x | React フレームワーク・ルーティング・SSR |
| **React** | 19.x | UI ライブラリ |
| **TypeScript** | 5.x | 型安全な開発 |
| **Tailwind CSS** | 4.x | スタイリング |

---

## 開発ツール

| 技術 | 用途 |
|------|------|
| **ESLint** | コード品質・リンター |
| **eslint-config-next** | Next.js 用 ESLint 設定 |

---

## 技術選定の理由（案）

- **Next.js**: フルスタック開発が可能、Vercel へのデプロイが容易
- **React 19**: 最新の React 機能を活用
- **TypeScript**: シミュレーション計算の型安全性を確保
- **Tailwind CSS**: 迅速な UI 開発、一貫したデザイン

---

## 採用技術（確定）

| 項目 | 採用 | 理由 |
|------|------|------|
| データ永続化 | **localStorage** | 投資計画の保存に十分、実装がシンプル |
| グラフ・チャート | **Recharts** | React との相性が良く、評価額推移の可視化に適する |
| 状態管理 | **React useState / useReducer** | 初期規模では十分、必要に応じて Context を追加 |

---

## 環境・実行

```bash
# 開発サーバー起動
npm run dev

# ビルド
npm run build

# 本番起動
npm run start
```
